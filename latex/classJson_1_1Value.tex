\doxysubsection{Detailed Description}
Represents a \href{http://www.json.org}{\texttt{ JSON}} value. 

This class is a discriminated union wrapper that can represents a\+:
\begin{DoxyItemize}
\item signed integer \mbox{[}range\+: \doxylink{classJson_1_1Value_adf754fa3bfc9897ac4b5158039b25d9f}{Value\+::min\+Int} -\/ \doxylink{classJson_1_1Value_ab6042b06093ce9871db116aa8b8e9c90}{Value\+::max\+Int}\mbox{]}
\item unsigned integer (range\+: 0 -\/ \doxylink{classJson_1_1Value_afed7d67975ffa7852d02f245c280a6b7}{Value\+::max\+UInt})
\item double
\item UTF-\/8 string
\item boolean
\item \textquotesingle{}null\textquotesingle{}
\item an ordered list of \doxylink{classJson_1_1Value}{Value}
\item collection of name/value pairs (javascript object)
\end{DoxyItemize}

The type of the held value is represented by a \doxylink{namespaceJson_a7d654b75c16a57007925868e38212b4e}{Value\+Type} and can be obtained using \doxylink{classJson_1_1Value_a8ce61157a011894f0252ceed232312de}{type()}.

Values of an \doxylink{namespaceJson_a7d654b75c16a57007925868e38212b4eae8386dcfc36d1ae897745f7b4f77a1f6}{object\+Value} or \doxylink{namespaceJson_a7d654b75c16a57007925868e38212b4eadc8f264f36b55b063c78126b335415f4}{array\+Value} can be accessed using \doxylink{classJson_1_1Value_a7d99f5dba388cdaa152ce6ef933d64ef}{operator\mbox{[}$\,$\mbox{]}()} methods. Non-\/const methods will automatically create the a \doxylink{namespaceJson_a7d654b75c16a57007925868e38212b4ea7d9899633b4409bd3fc107e6737f8391}{null\+Value} element if it does not exist. The sequence of an \doxylink{namespaceJson_a7d654b75c16a57007925868e38212b4eadc8f264f36b55b063c78126b335415f4}{array\+Value} will be automatically resized and initialized with \doxylink{namespaceJson_a7d654b75c16a57007925868e38212b4ea7d9899633b4409bd3fc107e6737f8391}{null\+Value}. \doxylink{classJson_1_1Value_a7a064d8aa47fde09a268be2aea992134}{resize()} can be used to enlarge or truncate an \doxylink{namespaceJson_a7d654b75c16a57007925868e38212b4eadc8f264f36b55b063c78126b335415f4}{array\+Value}.

The \doxylink{classJson_1_1Value_a034eb7bf85a44fa759bdaa232788ca66}{get()} methods can be used to obtain default value in the case the required element does not exist.

It is possible to iterate over the list of member keys of an object using the \doxylink{classJson_1_1Value_a5fa4e5279e83a421f9c128dd78be652b}{get\+Member\+Names()} method.

\begin{DoxyNote}{Note}
\doxylink{classJson_1_1Value_ada6ba1369448fb0240bccc36efaa46f7}{Value} string-\/length fit in size\+\_\+t, but keys must be \texorpdfstring{$<$}{<} 2\texorpdfstring{$^\wedge$}{\string^}30. (The reason is an implementation detail.) A \#\+Char\+Reader will raise an exception if a bound is exceeded to avoid security holes in your app, but the \doxylink{classJson_1_1Value}{Value} API does {\itshape not} check bounds. That is the responsibility of the caller. 
\end{DoxyNote}

Constructs a value from a static string. 

Like other value string constructor but do not duplicate the string for internal storage. The given string must remain alive after the call to this constructor.

\begin{DoxyNote}{Note}
