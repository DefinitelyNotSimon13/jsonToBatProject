\doxysubsection{Detailed Description}
\doxylink{classjson_1_1JSONHandler}{JSONHandler} class. 

The \doxylink{classjson_1_1JSONHandler}{JSONHandler} class is responsible for parsing a json file and creating a \doxylink{classjson_1_1JSONData}{JSONData} object from it when requested. It assigns all necessary values to the \doxylink{classjson_1_1JSONData}{JSONData} object. Most of the error handling is done in the \doxylink{classjson_1_1JSONData}{JSONData} object.
